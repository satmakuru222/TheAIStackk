## How To Use The n8n Workflows

1. Open n8n.
2. Go to Workflows.
3. Select Import from File.
4. Choose one JSON file from `n8n-workflows/`.
5. Configure the missing credentials highlighted by n8n.
6. Update IDs such as spreadsheet IDs, database IDs, Slack channel IDs, and publication IDs.
7. Test the happy path with the workflow test payload.
8. Test edge cases.
9. Activate the workflow only after the test run passes.
